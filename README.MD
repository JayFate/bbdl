# bbdl

命令行视频下载工具，同时支持哔哩哔哩, 抖音, YouTube, P站视频下载。

下载外网视频请自备科学上网环境。

### 安装

全局安装 bbdl

```sh
npm install -g bbdl
```

### 使用

```sh
bbd https://www.bilibili.com/video/BV12W411j7cH https://v.douyin.com/6oSqeNY/ https://www.pornhub.com/view_video.php?viewkey=ph613c8cc6203e8
```

视频下载后保存在 `<cwd>/bbdl` 目录
